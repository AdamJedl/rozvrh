# Rozvrh

Aplikace je nasazena na [rozvrh.matousvolf.cz](https://rozvrh.matousvolf.cz/).

HTML [veřejného rozvrhu](https://delta-skola.bakalari.cz/Timetable/Public) na stránkách školy.
